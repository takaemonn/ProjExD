# 第2回
## 電卓(ex02.calc.py)
### 追加機能
- 四則演算：＋以外の四則演算をする
- クリア：：entryに入力されている数字，数式の1文字をdeleteする
- オールクリア：entryに入力されている数字，数式の文字列全体をdeleteする
- 階乗計算：入力された数字の階乗を計算する(ただし自然数に限る)
- 総和計算：１から入力された値までの和を計算する(ただし自然数に限る)
